Répartition des tâches.

  • Clément

Clément a réalisé l'ensemble du code HTML et CSS, afin de créer l'interface utilisateur et l'aspect visuel du site. Il a fallu mettre en place des menus à gauche ou en bas de page afin de permettre une navigation facile et rapide, ainsi que des boutons de retour, afin de pouvoir revenir au menu principal. Aussi, il a réalisé les boutons interractifs de la page d'accueil, afin de permettre aux utilisateurs de choisir leurs actions. Une fois les actions envoyées, c'est le script Python qui les prend en charge, mais ça, c'est le rôle d'un autre membre de l'equipe...

  • Ugo

Ugo s'est chargé d'écrire le script Python utilisant Flask, afin de traiter les requêtes en voyées depuis le site. Un partie de son travail est visible dans le code HTML, là où il s'agit d'effectuer une liaison entre plusieurs fichiers. Flask utilise python pour gérer des commandes SQL, afin de pouvoir agir sur la base de données, en ajoutant, modifiant et supprimmant des éléments par exemple. Or, celà à été le travail d'un autre membre du groupe...

  • Jao

Jao à agi au plus "bas" niveau de notre projet, en créant les bases de données contenant toutes les informations des utilisateurs. Tout ce que ces bases reçoivent de Flask sont stockées et prêtes à être affichées en cas de recherche de la part d'un utilisateur. Il a fallu effectuer entre le code SQL et le code Python afin de rendre les nombreuses fonctionnalités de notre site possible.
